Problem 7 – Pass/Fail Predictor (Machine Learning)
This project is a simple machine learning application that predicts whether a student will pass or fail based on their marks. It reads a dataset file that contains student scores, then trains a model to learn patterns from the data. After training, the program can calculate its accuracy and also predict results for new students when the user enters marks. It includes a menu system to train the model, check accuracy, make predictions, and generate a graph. The program is designed in a modular way using classes and separate files, and it validates user input so it does not crash. This project demonstrates practical use of machine learning, data processing, and clean program structure in a simple and understandable way.
