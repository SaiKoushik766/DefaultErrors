Problem 1 – Student Record Management (CRUD)
This project is a simple terminal-based system that helps manage student records like a small digital register. It allows the user to add new student details, view all saved students, update existing information, and delete records when needed. The program is built using a menu system so the user can choose options easily. It uses a class to represent each student and stores all data inside a file so the information is saved even after the program is closed. It also checks user input to prevent errors and keeps the output clean and easy to read. This project shows basic programming concepts like object-oriented programming, file handling, and structured logic.
